# مشروع ميثاق

## التقنيات
- Next.js
- Supabase
- Vercel
- Tailwind CSS

## ما تم إنجازه
- صفحة الهبوط الرئيسية
- إنشاء حساب
- تسجيل الدخول
- لوحة التحكم
- إنشاء عقد جديد
- عرض قائمة العقود
- فلترة العقود حسب الحالة
- صفحة تفاصيل العقد
- إرسال رابط العقد عبر واتساب
- صفحة توقيع العميل
- تحديث حالة العقد إلى موقع
- إنهاء العقد وتحويله إلى مكتمل
- تحميل العقد PDF عبر الطباعة

## حالات العقد
- draft = مسودة
- sent = تم الإرسال
- signed = موقع
- completed = مكتمل
